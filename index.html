Usage

Simply download the framework, extract it to the folder you would like to keep it in.run CreateApp.php create your application, like below: 

php /your_path/Toknot/Tool/Create.php

the script will create application file structure and write some base code

More info see ReadMe

特性

  • 纯面向对象编程模式
  • MVC设计模式
  • 数据库访问对象，查询构建，Active Record 数据库设计模型来实现数据库的灵活访问
  • 多样灵活的路由控制器，根据开发习惯来定制路由表
  • 由框架来驱动WEB应用，实现模块化编程
  • REST开发支持
  • 类UNIX的用户权限控制
  • PHP伪多重继承实现
  • String and Array, File type of Object implemented and iterator support, and provided most PHP string function smae name of methid
  • Very simple template of view syntax and expression like PHP
  • User model like unix user access control implemented
  • Admin model provided site admin manage site of developement framework
  • Process model provided command line process manage
  • Provided a javascript developement library and it lightweight, only a objec of javascript, fit programer

Requirements

ToKnot requires PHP 5.3 or later
