Usage

Simply download the framework, extract it to the folder you would like to keep it in，and include Toknot/Control/Application.php on your application index.php file, like below code:

your application of index.php:

use Toknot\Control\Application;

require_once '/your_install_path/Toknot/Control/Application.php';

$app = new Application;

$app->run('\Shop',dirname(__DIR__));

More info see ReadMe

Features

  • Model-View-Controller (MVC) design pattern
  • Database Access Objects (DAO), Query Builder, Active Record
  • Skinning and theming
  • Error handling
